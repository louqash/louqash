# Hello there 👋

## About me
I am a systems and performance engineer focusing on hard, mathematical problems that must run fast on real hardware — the kind you think through on paper before writing any code.

Most of my work lives where algorithms meet the metal: hand-tuned SIMD in hard real-time C++, squeezing neural networks onto custom accelerators, and writing low-level code where numerical correctness matters as much as execution speed.

Lately, I'm heading further down the stack — into high-performance C++ and compute kernels — while pursuing an M.Sc. in Quantum Information Technology.
